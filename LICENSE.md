**Cause:** `FarmDelay` set too low, triggering a game-side timeout.
**Fix:** Increase `FarmDelay` to at least `3.0` in the config block.

### ❌ Silent Aim Not Locking On

**Cause:** `AimFOV` radius is too small for current server density.
**Fix:** Increase `AimFOV` to `150–200` in config and re-execute.

**Cause:** `AllCars` flag is set to `false` in config.

### ❌ Script Crashes on Mobile

**Cause:** Insufficient RAM or overlay permission not granted.
**Fix:** Close background apps, re-grant overlay permission, relaunch.

### ❌ Server Hop Triggering Too Frequently

**Cause:** `HopThreshold` is set to `1` or `2` — too sensitive.
**Fix:** Raise `HopThreshold` to `4` or `5` for more stable sessions.

### ❌ Money Farm Not Accumulating

**Fix:** Enable both `MoneyFarm` and `AutoRob` simultaneously.
